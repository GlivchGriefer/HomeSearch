# Craigslist

> [!info] Source Profile
> **Type:** Private Rentals Dominant | **AI Searchable:** Yes | **Listing Quality:** Variable — vet carefully

---

## Search URL Template

```
https://{city}.craigslist.org/search/apa?min_price={min}&max_price={max}&min_bedrooms={beds}
```

**Example — Denver, 2br, $1,200–$1,900:**
```
https://denver.craigslist.org/search/apa?min_price=1200&max_price=1900&min_bedrooms=2
```

**Full parameter list:**
```
?min_price=     Max rent
?max_price=     Max rent
?min_bedrooms=  Minimum bedrooms
?max_bedrooms=  Maximum bedrooms
?cats_ok=1      Cats allowed
?dogs_ok=1      Dogs allowed
?is_furnished=1 Furnished
?laundry=1      In-unit laundry
?parking=1      Parking included
```

## Scam Red Flags

> [!warning] Watch For
> - [ ] Price far below market — classic bait
> - [ ] Owner "abroad" or can't meet in person
> - [ ] Asks for deposit before viewing
> - [ ] Duplicate photos found via reverse image search
> - [ ] Requests wire transfer or gift cards
> - [ ] No phone number, only email

---

## Notes

- [p] Best source for private landlords and small operators
- [p] Lower application fees and more flexible terms common
- [p] Negotiation often possible (first/last vs deposit)
- [c] Higher scam rate — always verify before sending anything
- [c] No standardized listing format — requires more manual review
- [c] Listings expire quickly, check frequently
