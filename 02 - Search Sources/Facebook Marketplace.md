# Facebook Marketplace

> [!info] Source Profile
> **Type:** Private Rentals | **AI Searchable:** No (login required) | **Listing Quality:** Variable

---

## How to Search

1. Go to [Facebook Marketplace → Rentals](https://www.facebook.com/marketplace/category/propertyrentals)
2. Set location to your target area and radius
3. Filter by price, beds, property type
4. Message directly through Messenger — faster response than email

---

## Key Filters Available

| Filter | Notes |
| :----- | :---- |
| Location + radius | Set miles from a point |
| Price | Monthly range |
| Beds | Min/max |
| Property type | Apartment, house, condo, townhouse |

---

## Tips

- [p] Private landlords = more flexibility on terms and timing
- [p] Seller profile visible — can vet the landlord's history
- [p] Messenger makes follow-up easy and documented
- [p] Listings often appear before hitting major sites
- [c] Cannot be searched by AI — manual only
- [c] No standardized listing format
- [c] Scams exist — always tour before committing
- [i] Search Facebook Groups for your city: "[City] Rentals" or "[City] Housing" for additional private listings
