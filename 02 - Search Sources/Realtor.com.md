# Realtor.com

> [!info] Source Profile
> **Type:** Rentals + For-Sale (MLS) | **AI Searchable:** Yes | **Listing Quality:** High — MLS-accurate

---

## Search URL Template

```
https://www.realtor.com/apartments/{city}_{state}/
```

**Example — Denver rentals:**
```
https://www.realtor.com/apartments/Denver_CO/
```

| Filter | Notes |

| Beds / Baths | Min count |

| Pets | Yes/no |
| Parking | Included |
