# Apartments.com

> [!info] Source Profile
> **Type:** Rentals Only | **AI Searchable:** Yes | **Listing Quality:** High — deep filter options

---

## Search URL Template

```
https://www.apartments.com/{city}-{state}/{beds}-bedrooms-under-{max_rent}/
```

**Example — Denver, 2br, under $2,000:**
```
https://www.apartments.com/denver-co/2-bedrooms-under-2000/
```

---

## Key Filters Available

| Filter | Notes |
| :----- | :---- |
| Price | Min/max monthly rent |
| Beds / Baths | Exact count |
| Property type | Apartment, house, condo, townhouse, basement |
| Amenities | In-unit W/D, dishwasher, A/C, balcony, gym, pool |
| Parking | Yes/no, type |
| Pets | Cats/dogs, size limits |
| Move-in date | Available by date |
| Income restrictions | Affordable housing |
| Rating | Resident reviews |
